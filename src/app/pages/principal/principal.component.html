INICIO CHAT LOGIN REGISTRO HISTORIAL

Descripción del Proyecto

Este proyecto tiene como objetivo ofrecer un sistema de registro y login para los usuarios. Además, cuenta con un sistema de chat dividido en grupos de trabajo. Los usuarios podrán enviar mensajes que se guardarán diariamente y se podrán consultar según el día deseado a través de un calendario. Estos mensajes pueden contener códigos e incluso archivos adjuntos.
